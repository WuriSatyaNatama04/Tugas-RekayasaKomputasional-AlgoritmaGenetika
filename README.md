# Tugas Rekayasa Komputasional - Algoritma Genetika

**Implementasi Genetic Algorithm (GA) untuk Pencarian Kata dalam Kamus Bahasa Daerah Luwu**

## Deskripsi
Program ini menggunakan Algoritma Genetika untuk mencari kata dalam kamus bahasa daerah Luwu dengan pendekatan:
- Representasi kromosom
- Fitness function (jumlah huruf yang cocok)
- Seleksi Roulette Wheel
- One-point Crossover
- Mutasi

## Cara Menjalankan
```bash
python ga_kamus.py
